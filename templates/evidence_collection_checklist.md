# Evidence Collection Checklist

## Chain of Custody
- Evidence ID:
- Collected by:
- Date/time collected:
- Stored location:

## What to Collect
- Screenshots of alerts and ransom notes
- Email headers (if phishing/BEC)
- System logs (Windows Event Logs / Syslog)
- EDR/SIEM alerts and telemetry
- Authentication logs (Okta/Azure AD/etc.)
- Network logs (Firewall/DNS/Proxy)

## IOC Documentation
- Domains
- IP addresses
- File hashes (MD5/SHA256)
- Filenames and paths

## Notes
Always preserve evidence before remediation steps.
